#### 프로젝트 명
    국내 친환경 자동차 비교분석과 수소차 지원 정보 서비스


#### 프로젝트 목표
1. 지역별 수소차 소유자의 관리 편리성 및 근접성 확보
2. 충전소 위치 정보 확인을 통해 수소차 방전 위험성 감소
3. 정부의 친환경 정책에 대비하여 수소차 구매 독려 및 인프라 확충
